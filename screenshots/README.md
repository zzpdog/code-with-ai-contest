# 运行截图说明

由于 Streamlit 应用需要浏览器 WebSocket 连接才能完整渲染，在无头环境中无法直接截图。

## 如何查看实际运行效果

### 本地运行
```bash
pip install -r requirements.txt
streamlit run app.py
```

浏览器将自动打开 http://localhost:8501

### 主要功能界面

1. **主界面** - 包含顶部指标卡片、2D/3D 地图切换
2. **侧边栏** - 频段筛选、终端类型筛选、RSRP/SINR 范围滑动条
3. **数据概览** - 频段分布柱状图、终端类型饼图
4. **深度分析** - RSRP 分布直方图、各频段速率对比

比赛要求提供 2-3 张 Web 应用运行截图，展示：
- 地图和侧边栏交互
